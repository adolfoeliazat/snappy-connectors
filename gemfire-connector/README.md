#SnappyData GemFire Connector

SnappyData GemFire Connector let's you connect SnappyData to GemFire, expose GemFire regions as Spark RDDs, save Spark RDDs to GemFire and execute GemFire OQL queries in your SnappyData applications
and expose the results as DataFrames.

##Features:

 - Expose GemFire region as Spark RDD with GemFire server-side filtering
 - RDD join and outer join GemFire region
 - Save Spark RDD to GemFire
 - Save DStream to GemFire
 - Execute GemFire OQL and return DataFrame

##Version and Compatibility

SnappyData GemFire Connector supports Spark 2.0
